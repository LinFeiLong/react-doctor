# react-doctor-v2

Scaffold for the React Doctor v2 rewrite.

This package currently exposes an advanced `createReactDoctor()` SDK client from `src/sdk/index.ts`, a thin `inspectReactProject()` convenience wrapper, deprecated v1-compatible APIs from `src/sdk/compat.ts`, and a placeholder `react-doctor` CLI entrypoint so the rewrite can grow behind a separate workspace package.
